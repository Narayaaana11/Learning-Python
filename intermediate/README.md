# Intermediate Python

This directory contains intermediate Python concepts and examples.

## Topics Covered

### 1. Object-Oriented Programming (`oop_basics.py`)
- Classes and objects
- Inheritance
- Encapsulation
- Class and instance attributes
- Methods and special methods

### 2. List Comprehensions and Lambda Functions (`comprehensions_lambda.py`)
- List comprehensions
- Dictionary comprehensions
- Set comprehensions
- Lambda functions
- map(), filter(), and sorted() with lambdas

## Running the Examples

```bash
python intermediate/oop_basics.py
python intermediate/comprehensions_lambda.py
```

## Practice Exercises

1. Create a class hierarchy for shapes (Circle, Rectangle, Triangle)
2. Implement a simple inventory system using OOP
3. Use list comprehensions to solve data transformation problems
4. Practice writing lambda functions for sorting and filtering
