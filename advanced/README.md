# Advanced Python

This directory contains advanced Python concepts and examples.

## Topics Covered

### 1. Decorators (`decorators.py`)
- Basic decorators
- Decorators with arguments
- functools.wraps
- Timing decorators
- Class-based decorators

### 2. Context Managers (`context_managers.py`)
- Using built-in context managers
- Creating custom context managers (class-based)
- Creating context managers with @contextmanager
- Multiple context managers

## Running the Examples

```bash
python advanced/decorators.py
python advanced/context_managers.py
```

## Practice Exercises

1. Create a caching decorator
2. Build a logging decorator that logs function calls
3. Create a context manager for database transactions
4. Implement a retry decorator for handling failures
