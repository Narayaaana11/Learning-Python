# Python Basics

This directory contains fundamental Python concepts and examples.

## Topics Covered

### 1. Hello World (`hello_world.py`)
- Basic print statements
- String formatting
- First Python program

### 2. Variables and Data Types (`variables_datatypes.py`)
- Numeric types (int, float, complex)
- Strings
- Booleans
- Collections (lists, tuples, dictionaries, sets)
- Type conversion

### 3. Control Flow (`control_flow.py`)
- If-else statements
- For loops
- While loops
- Break and continue
- Nested loops

### 4. Functions (`functions.py`)
- Defining functions
- Parameters and return values
- Default parameters
- *args and **kwargs
- Multiple return values

## Running the Examples

```bash
# Run any example
python basics/hello_world.py
python basics/variables_datatypes.py
python basics/control_flow.py
python basics/functions.py
```
